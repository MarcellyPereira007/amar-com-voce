## Sobre o projeto

Esse projeto nasce do desejo de transformar uma carta em uma experiência imersiva. A interface guia a leitura com um fluxo onde a própria tela reage à presença de quem lê.

O design une texturas analógicas com tecnologias modernas de desenvolvimento front-end.

## Tecnologias

A arquitetura foi pensada para entregar alta performance com animações complexas.
